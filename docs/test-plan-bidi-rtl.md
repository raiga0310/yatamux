## テスト計画: BiDi RTL 制御文字の幅0処理 (C-6)

### TC-C6-01: RIGHT-TO-LEFT MARK (U+200F) はカーソルを進めない
- **前提**: 80x24 のグリッド、カーソルが (0,0)
- **操作**: U+200F を含む文字列 `"A\u{200F}B"` を VtProcessor でフィード
- **期待結果**: カーソルが col=2 (A と B の分だけ進む)、U+200F はカーソルを進めない
