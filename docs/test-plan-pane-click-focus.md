## テスト計画: クリックによるペインフォーカス（F-9）

### TC-01: 単一ペインをクリック → フォーカス変化なし
- **前提**: ペインが1枚のみ
- **操作**: 任意の位置でマウス左クリック
- **期待結果**: active pane は変わらず、クラッシュしない

### TC-02: 垂直分割 — 右ペインをクリック → フォーカス右ペインへ
- **前提**: 左右2ペインに垂直分割（左がアクティブ）
- **操作**: 右ペイン領域をマウス左クリック
- **期待結果**: active が右ペインに切り替わる

### TC-03: 垂直分割 — 左ペインをクリック → フォーカス左ペインへ
- **前提**: 左右2ペインに垂直分割（右がアクティブ）
- **操作**: 左ペイン領域をマウス左クリック
- **期待結果**: active が左ペインに切り替わる

### TC-04: 水平分割 — 下ペインをクリック → フォーカス下ペインへ
- **前提**: 上下2ペインに水平分割（上がアクティブ）
- **操作**: 下ペイン領域をマウス左クリック
- **期待結果**: active が下ペインに切り替わる

### TC-05: セパレーター領域をクリック → フォーカス変化なし（またはいずれかのペイン）
- **前提**: 左右2ペインに垂直分割
- **操作**: セパレーター（1px 幅）上をクリック
- **期待結果**: クラッシュせず、フォーカスが変わらないか既存ペインのどちらかになる

### TC-06: ペイン外（パディング領域）をクリック → フォーカス変化なし
- **前提**: 任意のペイン構成
- **操作**: ウィンドウ端のパディング領域（PADDING_X/Y 内側）をクリック
- **期待結果**: フォーカスは変わらない（ヒットなし）

### TC-07: クリック後のスクロールオフセットリセット
- **前提**: 左右2ペインに垂直分割、右ペインでスクロールバック中
- **操作**: 右ペインをクリック（フォーカス移動が発生する場合）
- **期待結果**: scroll_offset が 0 にリセットされる

### TC-08: LayoutNode::pane_at_point — 単体テスト（ユニット）
- **操作**: `LayoutNode::pane_at_point(x, y, root)` を直接呼び出す
- **期待結果**:
  - 垂直分割: x < 半幅 → first_id, x ≥ 半幅+sep → second_id
  - 水平分割: y < 半高 → first_id, y ≥ 半高+sep → second_id
  - 範囲外: None
