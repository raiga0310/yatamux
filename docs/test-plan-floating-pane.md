## テスト計画: フローティングペイン (C-3)

### TC-01: floating_rect — コンテンツ中央に 80% 矩形
- **種別**: ユニットテスト
- **操作**: `PaneStore::floating_rect(PaneRect { x:0, y:0, w:200, h:100 })`
- **期待結果**: `w=160, h=80, x=20, y=10`

### TC-02: floating_rect — コンテンツ幅が奇数でも中央揃え
- **種別**: ユニットテスト
- **操作**: `PaneStore::floating_rect(PaneRect { x:0, y:0, w:101, h:51 })`
- **期待結果**: `w=80, h=40, x≥10, y≥5`（切り捨て誤差許容）

### TC-03: フローティングペイン表示時 active がフローティング ID に変わる（ユニットテスト）
- **前提**: `PaneStore { floating: Some(PaneId(2)), floating_visible: false, active: PaneId(1) }`
- **操作**: `store.show_float()`
- **期待結果**: `store.active == PaneId(2)`, `store.pre_float_active == Some(PaneId(1))`

### TC-04: フローティングペイン非表示時 active が元に戻る（ユニットテスト）
- **前提**: `store.show_float()` 後
- **操作**: `store.hide_float()`
- **期待結果**: `store.active == PaneId(1)`, `store.floating_visible == false`

### TC-05: レイアウト pane_ids にフローティングペインは含まれない（ユニットテスト）
- **前提**: `LayoutNode::Leaf(PaneId(1))`, `PaneStore { floating: Some(PaneId(99)) }`
- **操作**: `store.layout.pane_ids()`
- **期待結果**: `[PaneId(1)]`（PaneId(99) は含まれない）

### TC-06: Ctrl+F でフローティングペインが作成・表示される（手動テスト）
- **前提**: フローティングペインなし（初回）
- **操作**: Pane モードで `F` を押す
- **期待結果**: 中央にオーバーレイペインが表示され、フォーカスがそこに移る

### TC-07: 既存フローティングペインを Ctrl+F で非表示にする（手動テスト）
- **前提**: フローティングペイン表示中
- **操作**: Pane モードで `F` を押す
- **期待結果**: フローティングペインが非表示になり、元のペインにフォーカスが戻る

### TC-08: フローティングペインを再度 Ctrl+F で再表示（手動テスト）
- **前提**: フローティングペイン非表示
- **操作**: Pane モードで `F` を押す
- **期待結果**: 同じフローティングペインが再表示され、フォーカスが移る

### TC-09: フローティングペインに枠線が描画される（手動テスト）
- **期待結果**: フローティングペインの境界に枠線（2px）が表示される

### TC-10: フローティングペイン表示中の通常ペイン操作（手動テスト）
- **前提**: フローティングペイン表示中
- **操作**: クリックでフローティング外の通常ペインをクリック
- **期待結果**: フローティングペインが非表示になり、クリックしたペインにフォーカス
