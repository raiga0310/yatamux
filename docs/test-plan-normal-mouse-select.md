## テスト計画: Normal モードでのマウス範囲選択 + Ctrl+C コピー (C-17)

### TC-01: ドラッグで単一行を選択する

- **前提**: Normal モード。ペインに何らかのテキストが表示されている。
- **操作**: アクティブペイン内でマウス左ボタンを押し、同じ行の別セルまでドラッグしてボタンを離す。
- **期待結果**: ドラッグした範囲のセルが選択ハイライト（前景・背景反転）で表示される。

### TC-02: ドラッグで複数行を選択する

- **前提**: Normal モード。ペインに複数行のテキストが表示されている。
- **操作**: アクティブペイン内でマウス左ボタンを押し、異なる行まで縦にドラッグしてボタンを離す。
- **期待結果**: 開始行から終了行まで、terminal 慣例に従ったテキスト選択ハイライトが表示される
  （開始行は開始列から行末まで、中間行は全幅、終了行は行頭から終了列まで）。

### TC-03: 選択後 Ctrl+C でクリップボードにコピーされる

- **前提**: Normal モードで TC-01 または TC-02 の操作をして選択範囲がある状態。
- **操作**: Ctrl+C を押す。
- **期待結果**: 選択テキストがシステムクリップボードにコピーされる。PTY には \x03 が送信されない。
  選択ハイライトが消える。

### TC-04: 選択なしの Ctrl+C は PTY に \x03 を送信する

- **前提**: Normal モード。選択範囲なし（または単なるクリックのみ）。
- **操作**: Ctrl+C を押す。
- **期待結果**: 従来通り \x03 が PTY に送信される。実行中のプロセスが中断される。

### TC-05: 新規クリックで選択範囲がクリアされる

- **前提**: Normal モードで選択範囲がある状態。
- **操作**: 選択範囲外を左クリックする。
- **期待結果**: 選択ハイライトが消える。

### TC-06: 別ペインクリックで選択がクリアされてフォーカスが移動する

- **前提**: Normal モードで選択範囲がある状態。複数ペイン存在。
- **操作**: 別のペインを左クリックする。
- **期待結果**: 選択ハイライトが消え、クリックしたペインにフォーカスが移動する。

### TC-07: Copy モード（V）は従来通り動作する

- **前提**: Pane モード（Ctrl+B）→ V でコピーモードに入った状態。
- **操作**: hjkl でカーソル移動、v で選択、y/Enter でヤンク。
- **期待結果**: C-12 の既存動作が変わらず機能する。C-17 の変更による影響がない。

### TC-08: 選択範囲の `is_in_normal_selection` ロジック（単体）

- **前提**: `is_in_normal_selection` 関数に対するユニットテスト。
- **操作**: 単一行選択（col=2, row=1, col=5, row=1）に対して:
  - (1, 1): false（範囲外）
  - (2, 1): true（境界）
  - (3, 1): true（内側）
  - (5, 1): true（境界）
  - (6, 1): false（範囲外）
  - (3, 0): false（別行）
- **期待結果**: 各ケースが期待通りの bool を返す。

### TC-09: 複数行選択の `is_in_normal_selection` ロジック（単体）

- **前提**: `is_in_normal_selection` 関数に対するユニットテスト。
- **操作**: 複数行選択（anchor=(2, 1), end=(4, 3)）に対して:
  - (1, 1): false（開始行・列より前）
  - (2, 1): true（開始行・開始列）
  - (0, 2): true（中間行・全幅）
  - (4, 3): true（終了行・終了列）
  - (5, 3): false（終了行・列より後）
- **期待結果**: 各ケースが期待通りの bool を返す。
